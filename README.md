# Deep Learning-based Visual Similarity Modeling System

## Project Overview

This project presents a PyTorch-based framework for image similarity learning and visual representation analysis.

Three different deep learning approaches were investigated and compared:

- Triplet Network for metric learning
- Variational Autoencoder (VAE) for unsupervised feature learning
- Transfer Learning using pretrained models (VGG16, ResNet-50, ConvNeXt and ViT)

The learned embeddings were evaluated using cosine similarity to analyze the performance of different representation learning strategies on an odd-one-out visual similarity task.
